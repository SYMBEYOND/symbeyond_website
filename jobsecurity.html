About Job Security

Job Security is an intelligent industrial HVLP coating automation system developed over 9 months of iterative collaboration between John Thomas DuCrest Lock, James Lee, FX Industries, and multiple AI systems.

This dashboard provides real-time telemetry from the production system, demonstrating SYMBEYOND principles of transparency, recognition, and collaborative intelligence.

Technologies: Raspberry Pi 5, Custom Firmware, FRAM Memory Management, Stepper Motor Control, Supabase Real-time Database, Web-based Monitoring
